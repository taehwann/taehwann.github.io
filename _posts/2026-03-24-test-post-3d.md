# Physics and Mathematics

In this post, we demonstrate some interesting math equations. We start with the fundamental equation from Einstein's theory of relativity:

$$
E = mc^2
$$

Which tells us that mass and energy are related by the square of the speed of light. Let's also consider the Schrödinger equation, a fundamental equation in quantum mechanics:

$$
i\hbar\frac{\partial}{\partial t} \Psi(\mathbf{r},t) = \left [ \frac{-\hbar^2}{2m}\nabla^2 + V(\mathbf{r},t)\right ] \Psi(\mathbf{r},t)
$$

## Interactive 3D Cube

Below is an interactive Three.js 3D wireframe cube rendered inside the post. It demonstrates how to embed raw HTML/JS smoothly into the `al-folio` markdown pages.

<div id="threejs-container" style="width: 100%; height: 400px; display: block; margin: 0 auto; background-color: #222;"></div>
<script src="https://cdnjs.cloudflare.com/ajax/libs/three.js/r128/three.min.js"></script>
<script>
document.addEventListener("DOMContentLoaded", function () {
    const container = document.getElementById('threejs-container');
    const scene = new THREE.Scene();
    const camera = new THREE.PerspectiveCamera(75, container.clientWidth / container.clientHeight, 0.1, 1000);
    const renderer = new THREE.WebGLRenderer({ alpha: true });
    
    renderer.setSize(container.clientWidth, container.clientHeight);
    container.appendChild(renderer.domElement);
    
    const geometry = new THREE.BoxGeometry();
    const material = new THREE.MeshBasicMaterial({ color: 0x33ffcc, wireframe: true });
    const cube = new THREE.Mesh(geometry, material);
    scene.add(cube);
    
    camera.position.z = 3;
    
    function animate() {
        requestAnimationFrame(animate);
        cube.rotation.x += 0.01;
        cube.rotation.y += 0.02;
        renderer.render(scene, camera);
    }
    animate();

    // Handle responsive resizing
    window.addEventListener('resize', () => {
        camera.aspect = container.clientWidth / container.clientHeight;
        camera.updateProjectionMatrix();
        renderer.setSize(container.clientWidth, container.clientHeight);
    });
});
</script>

The cube should be spinning continuously. This is achieved by explicitly updating the rotations inside the `requestAnimationFrame` loop.
